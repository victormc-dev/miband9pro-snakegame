# 贪吃蛇游戏

小米手环快应用版本的贪吃蛇小游戏，支持多屏适配，提供完整的游戏体验。

## 功能特性

- 🎮 经典贪吃蛇玩法
- ⚙️ 可调节游戏速度（5档：很慢、慢、正常、快、很快）
- 🍎 可调节食物大小（3档：小、中、大）
- 📱 完美适配 336×480 分辨率的小米手环
- 💾 设置自动保存，下次启动自动应用
- 🎯 触摸滑动控制方向
- 📊 实时得分显示

## 游戏说明

### 开始游戏

1. 在首页点击「开始游戏」进入游戏
2. 上下左右滑动屏幕控制蛇的移动方向
3. 吃到红色食物获得 10 分
4. 撞墙或撞到自己游戏结束

### 设置调整

1. 在首页点击「设置」进入设置页面
2. 使用 +/- 按钮调整游戏速度和食物大小
3. 设置会自动保存

### 页面导航

- 每个页面都有返回按钮可以返回上一页
- 开始页 → 游戏页
- 开始页 → 设置页

## 技术栈

- Xiaomi Vela JS 应用框架
- 快应用开发范式
- 响应式 UI 组件
- @system.storage - 本地存储
- @system.router - 页面路由

## 分辨率适配

- 设计分辨率：336×480
- 支持多屏自适应
- Flexbox 响应式布局
